# Dataset

## amazon_prime_titles.csv

Contains metadata for ~9 687 titles (movies and TV shows) available on Amazon Prime Video.

| Column | Description |
|---|---|
| `show_id` | Unique identifier |
| `type` | `Movie` or `TV Show` |
| `title` | Title of the content |
| `director` | Director(s) |
| `cast` | Main cast |
| `country` | Country of production |
| `date_added` | Date added to Amazon Prime |
| `release_year` | Original release year |
| `rating` | Content rating (e.g. PG-13, R) |
| `duration` | Duration in minutes (movies) or seasons (TV shows) |
| `listed_in` | Comma-separated genre/category tags |
| `description` | Short synopsis |

**Source:** [Kaggle – Amazon Prime Movies and TV Shows](https://www.kaggle.com/datasets/shivamb/amazon-prime-movies-and-tv-shows)
